# Transformer From Scratch (PyTorch)

A pure PyTorch implementation of the Transformer architecture from the "Attention Is All You Need" paper. 

This project is not just about writing the code, but about deeply understanding the underlying mechanics. It is built layer-by-layer with a strict focus on mathematical fundamentals and dimensional analysis.

## 🚀 Current Progress

The following components of the architecture have been successfully implemented from scratch:

1. Tokenization & Embedding
2. Positional Encoding
3. Scaled Dot-Product Attention
4. Multi-Head Attention (MHA)
